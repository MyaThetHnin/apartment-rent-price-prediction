# Apartment Rental Price Prediction

This project focuses on building a ML pipeline to predict apartment rental prices based on various housing features. The dataset was explored and cleaned before applying feature engineering techniques to improve model performance. Multiple ML algorithms were trained and fine-tuned, and their performance was evaluated and compared to determine the best predictive model.

## Technologies
- Python
- Pandas
- Scikit-learn
- Spark MLlib
- Jupyter Notebook

## Project Steps
- Data exploration
- Data preprocessing
- Feature engineering
- Model training
- Model evaluation
